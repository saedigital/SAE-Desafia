## Passo a passo para execução:


### AMBIENTE WINDOWS

Criar um ambiente com Virtualenv

```$virtualenv ENV```


Ativar a MV

``` $ENV\Script\activate ```


Instalar as Dependências

``` $pip install -r requirements.txt ```


Executar o projeto

```$python app.py```


Acessar http://localhost:5000


### AMBIENTE LINUX

Criar um ambiente com Virtualenv

``` $virtualenv ENV ```


Ativar a MV

```$source ENV/bin/activate```


Instalar as Dependências

```$pip install -r requirements.txt```


Executar o projeto

```$python app.py```


Acessar http://localhost:5000
